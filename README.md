# Outlive365 MVP

Static MVP web app for the Outlive365 PRD.

## Includes

- Account creation (prototype/local)
- 10-minute onboarding assessment
- Top 3 longevity priorities engine
- Daily habit dashboard (<=5 habits)
- Weekly adherence, streak, and 30-day consistency score
- Science brief layer with evidence badges

## Run

Open `index.html` in your browser.
